# Changelog

- Базовый релиз «Счётчика калорий».
- Дневник по приёмам (завтрак/обед/ужин/перекус) с подсчётом БЖУ/ккал.
- Каталог блюд: локальные и опубликованные (Supabase), сортировка по популярности/адекватности.
- Публикация блюд с фото и описанием, детальная карточка блюда.
- Авторизация/регистрация через Supabase, сохранение дневников и блюд на пользователя.
