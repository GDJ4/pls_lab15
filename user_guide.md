# User Guide — Счётчик калорий

## Установка
1) Установите приложение (AAB/APK/iOS build).  
2) При первом запуске: разрешите доступ к фотогалерее/камере, если планируете публиковать блюда.  
3) Войдите/зарегистрируйтесь (Supabase Auth) или продолжите в гостевом режиме (данные будут локальными).

## Быстрый старт
1) Вкладка «День»
   - Выберите дату (календарь в AppBar).
   - Нажмите «Добавить блюдо» в нужном приёме (завтрак/обед/ужин/перекус).
   - Выберите блюдо из каталога или создайте своё; укажите порцию.  
   - Итоговые калории/БЖУ за день отображаются в карточке «Итоги за день».

2) Создание блюда
   - Нажмите «Создать своё блюдо» или «Опубликовать своё блюдо» на вкладке «Блюда».
   - Заполните калории и БЖУ, при желании добавьте описание/рецепт.
   - При публикации добавьте фото — блюдо появится в общем каталоге (Supabase).

3) Каталог блюд
   - Вкладка «Блюда» → поиск по названию, подсказки популярных.
   - Тап по карточке открывает детали: фото, описание, БЖУ, кнопка «Добавить в дневник».
   - Свои блюда можно скрыть/опубликовать переключателем; без фото публикация не разрешена.

## Советы
- Для точных расчётов указывайте порцию (0.5, 1.0 и т.п.).
- Публикуйте адекватные значения калорий — популярность и сортировка учитывают «правдоподобность».
- Авторизуйтесь, чтобы синхронизировать блюда и дневники между устройствами.

## Технические требования
- Интернет обязателен только для авторизации и работы с публичными блюдами; офлайн сохраняются локальные данные.
- Разрешения: камера/галерея для фото блюд.
